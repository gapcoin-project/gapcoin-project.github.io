Elementary Gapcoin mining with default miner parameter settings

Pool mining with getwork

gapminer -o http://your-pool.club -p 4200 -u user.worker -x password -t 4

Pool mining with stratum (recommended)

gapminer -o your-pool.club -p 3385 -u user.worker -x password -t 4 --stratum

Solo mining

gapminer -o 127.0.0.1 -p 31397 -u <rpc user> -x <rpc password> -t 4
